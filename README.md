# KENTUCKY TRANSPORTATION NOISE EXPOSURE

An interactive web map exploring transportation-related noise exposure across Kentucky. 

## Live Map

→ https://jfltx.github.io/ky-noise-exposure

## Project Overview

This project visualizes broadly-modeled transportation noise alongside roadway traffic counts and key contextual features such as schools, libraries, airports, parks, and industrial sites.

The goal is to support exploratory analysis of potential noise exposure, particularly for noise sensitive land uses (schools, libraries, parks, & residential areas).

## How to Use

- Zoom in to explore the map at finer detail. 
- Click on roadways to view traffic counts and estimated noise exposure. 
- Click the map to view modeled noise values.

## Important Note

Distance-based noise estimates are simplified screening tools and may not align exactly with the modeled noise surface.  
These values are intended to help identify potentially noise-exposed areas and should not be used as measured noise levels.

## Data Sources

- Modeled Noise Surface → [U.S. DOT Bureau of Transportation Statistics](https://www.bts.gov/geospatial/national-transportation-noise-map)
- Traffic Counts → [KYTC ArcGIS REST Services](https://maps.kytc.ky.gov/arcgis/rest/services/Apps/TrafficCounts/MapServer/6)
- Schools, Libraries, Airports, Industrial Sites → [kygisserver WGS84WM_Services](https://kygisserver.ky.gov/arcgis/rest/services/WGS84WM_Services)
- Compiled Parks → &copy; OpenStreetMap contributors, [LFUCG](https://data.lexingtonky.gov/datasets/764044274e974dbba9069d9dab7dcb34_0) and [LOJIC](https://data.lojic.org/datasets/louisville-ky-metro-parks/about)
